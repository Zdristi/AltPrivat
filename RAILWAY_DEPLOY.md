# Деплой бота на Railway

## Подготовка

1. Зарегистрируйтесь на [Railway](https://railway.app/)
2. Создайте новый проект

## Способ 1: Через GitHub (рекомендуется)

1. Создайте публичный репозиторий на GitHub
2. Закиньте туда все файлы из этой папки
3. В Railway нажмите "New Project" → "Deploy from GitHub repo"
4. Выберите ваш репозиторий

## Способ 2: Через загрузку файлов

1. В Railway нажмите "New Project" → "Deploy from scratch"
2. Загрузите все файлы из этой папки

## Настройка переменных окружения

После деплоя перейдите во вкладку "Variables" и добавьте:

- BOT_TOKEN (ваш токен бота из @BotFather)
- CHANNEL_ID (ID канала, к которому даете доступ)
- ADMIN_USER_ID (ваш ID пользователя в Telegram)
- SUBSCRIPTION_NAME (название подписки)
- SUBSCRIPTION_DAYS (количество дней подписки)
- SUBSCRIPTION_PRICE (цена подписки)

## Запуск

После настройки переменных Railway автоматически запустит приложение.
Проверьте логи во вкладке "Logs", чтобы убедиться, что бот запустился без ошибок.

## Важно

- Railway предоставляет бесплатный тариф с ограничениями
- Приложение может "засыпать" при бездействии в течение 5 минут
- Бесплатные проекты ограничены по времени работы
