# Инструкция по загрузке файлов в репозиторий GitHub

## Подготовка перед загрузкой

1. Убедитесь, что у вас установлен Git:
   - Скачайте с https://git-scm.com/downloads
   - Перезапустите командную строку после установки

2. Откройте командную строку (или Git Bash) и перейдите в папку с проектом:
   ```bash
   cd "C:\Users\Andrey\Desktop\AltPrivat"
   ```

## Загрузка файлов в репозиторий

1. Инициализируйте локальный репозиторий:
   ```bash
   git init
   ```

2. Добавьте удаленный репозиторий (ваш GitHub репозиторий):
   ```bash
   git remote add origin https://github.com/Zdristi/AltPrivat.git
   ```

3. Добавьте все файлы к следующему коммиту:
   ```bash
   git add .
   ```

4. Сделайте первый коммит:
   ```bash
   git commit -m "Initial commit: добавлены все файлы бота"
   ```

5. Установите основную ветку как main:
   ```bash
   git branch -M main
   ```

6. Загрузите файлы в репозиторий на GitHub:
   ```bash
   git push -u origin main
   ```

## Возможные проблемы и решения

Если появляется ошибка аутентификации:
1. Сгенерируйте персональный токен на GitHub:
   - Перейдите в Settings > Developer settings > Personal access tokens > Tokens (classic)
   - Нажмите "Generate new token"
   - Выберите нужные разрешения и скопируйте токен
   - При выполнении команды git push используйте токен вместо пароля

## Проверка загрузки

После выполнения команд зайдите на https://github.com/Zdristi/AltPrivat
и убедитесь, что все файлы загружены в репозиторий.
